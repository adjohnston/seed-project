# seed-project

Seed project with
- webpack
- hot reloading
- Babel
- unit tests (Jasmine)
- Sass

## Installation and development

```
npm install && bower install
npm start
```

Then go to [http://localhost:8080/](http://localhost:8080/) to see it.

### Testing

```
npm test
```

This will run Jasmine tests with Karma.
